# Hangman Game (Python)

# Overview

This is a simple Hangman game written in Python. The program randomly selects a word, and the player has to guess it one letter at a time before reaching the maximum number of incorrect attempts.

# Features

* Random word selection
* Tracks guessed letters
* Limits incorrect guesses to 6
* Displays progress after each guess
* Beginner-friendly code

# Requirements

* Python 3.x

# How to Run

1. Download or clone this repository
2. Open terminal or command prompt
3. Run the program:

   ```
   python hangman.py
   ```

# How to Play

* Guess one letter at a time
* Correct guesses reveal the letter in the word
* Wrong guesses increase the count
* You lose after 6 incorrect guesses
* Guess the full word to win
