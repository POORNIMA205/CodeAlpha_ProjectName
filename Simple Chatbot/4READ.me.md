# Email Extractor using Python

# Project Description

This project is a simple Python script that extracts all email addresses from a text file and saves them into a separate output file. It helps automate the process of finding emails from large amounts of text data.

---

# Objective

- To extract valid email addresses from a given text file
- To store the extracted emails in a clean and organized format
- To automate a repetitive real-life task using Python

---

# Technologies Used

- Python
- Regular Expressions ("re" module)
- File Handling

---

# Project Structure

Email-Extractor/
│── input.txt        # File containing text with emails
│── emails.txt       # Output file with extracted emails
│── extractor.py     # Python script
│── README.md        # Project documentation

---

# How to Run the Project

1. Install Python on your system
2. Create a file named "input.txt" and add some text with emails
3. Run the Python script:
   python extractor.py
4. After execution, check the "emails.txt" file for results

---

# Output

The output file ("emails.txt") will contain:

- List of extracted email addresses
- Numbered format
- Total count of emails
