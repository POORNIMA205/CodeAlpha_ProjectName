# Stock Portfolio Tracker

# Overview

The Stock Portfolio Tracker is a simple Python application that allows users to calculate their total investment in stocks. The program takes stock names and quantities as input and uses predefined prices to compute the total value of the portfolio.

---

# Aim

To develop a basic Python program that helps users track their stock investments and understand fundamental programming concepts.

---

# Features

- Accepts multiple stock inputs from the user
- Uses a predefined dictionary for stock prices
- Calculates individual and total investment value
- Displays a clear portfolio summary
- Saves results into a text file ("portfolio.txt")

---

# Concepts Used

- Dictionary
- Loops ("for")
- Conditional statements ("if")
- Arithmetic operations
- File handling

---

# Technologies

- Python 3.x

---

# Project Files

- "main.py" → Main program
- "portfolio.txt" → Output file

---

# How to Run

1. Install Python (3.x)
2. Save the code in a file named "main.py"
3. Open terminal / command prompt
4. Run the command:
   python main.py
5. Enter stock details when prompted

---

# Input Example

Enter number of stocks: 2
Enter stock name: AAPL
Enter quantity: 3
Enter stock name: TSLA
Enter quantity: 2

---

# Output Example

--- Portfolio Summary ---
AAPL -> Quantity: 3, Price: 180, Value: 540
TSLA -> Quantity: 2, Price: 250, Value: 500

Total Investment Value: 1040

---

# Output File

The program creates a file named "portfolio.txt" which contains:

- Stock details
- Price and quantity
- Total investment value
