# imple Chatbot using Python

# Project Description

This project is a basic rule-based chatbot developed using Python. The chatbot responds to user inputs with predefined replies. It is designed to demonstrate simple concepts of conditional statements and loops.

---

# Features

* Responds to basic greetings like "hello"
* Answers simple questions like "how are you"
* Introduces itself when asked
* Exits the program when the user types "bye"
* Handles unknown inputs with a default response

---

# Technologies Used

* Python (Core Programming)
* Command Line Interface (CLI)

---

# How to Run the Program

1. Install Python on your system
2. Save the code in a file named `chatbot.py`
3. Open terminal or command prompt
4. Navigate to the file location
5. Run the program using:

```bash
python chatbot.py
```
